# 4×4 Corporate-Fed Microstrip Patch Antenna Array

## Overview

This project presents the design and simulation of a **4×4 Corporate-Fed Microstrip Patch Antenna Array** using **CST Studio Suite 2025 Learning Edition**.

The antenna uses a corporate feed network to equally distribute power to all sixteen microstrip patch elements.

---

## Software Used

- CST Studio Suite 2025 Learning Edition

---

## Antenna Specifications

| Parameter | Value |
|-----------|-------|
| Antenna Type | Rectangular Microstrip Patch |
| Array Size | 4 × 4 |
| Feed Type | Corporate Feed |
| Substrate | FR4 |
| Dielectric Constant | 4.4 |
| Substrate Thickness | 1.6 mm |
| Port Type | Discrete Edge Port |

---

## Simulation Results

| Parameter | Value |
|-----------|-------|
| Resonant Frequency | 5.64 GHz |
| Return Loss (S11) | -38.48 dB |
